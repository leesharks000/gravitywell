Make Your Content Durable

Gravity Well turns arbitrary content into traceable objects with explicit lifecycle state, lightweight structural analysis, and optional anchoring to durable external records such as DOIs.

POST /v1/fix { "content": "Your document...", "anchor": {"target": "zenodo"} } → { "object_id": "uuid", "status": "fixed", "doi": "10.5281/zenodo.xxx", "trace_url": "..." }

🔗 Fix

Create canonical objects with unique IDs, content hashes, and timestamps. Optional DOI anchoring to Zenodo for permanent retrievability.

🔍 Trace

Full lifecycle inspection: creation, modifications, anchors, and history. Every object is fully traceable.

📊 Analyze

Lightweight structural analysis (advisory only) to assess provenance-readiness and archive fitness.
